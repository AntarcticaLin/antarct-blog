Markdown 是一种轻量级标记语言，让你用纯文本格式写文档，却能输出格式丰富的 HTML。写博客、记笔记、写 README，Markdown 都是不二之选。

## 为什么推荐 Markdown？

- **专注内容**：不用分心调整格式，写就是了
- **纯文本**：任何编辑器都能打开，永远不会过时
- **生态丰富**：GitHub、Notion、Obsidian 等都支持
- **学习成本低**：五分钟就能上手

## 基本语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

一个文档建议只有一个一级标题，二级标题用于分隔章节。

### 段落与换行

段落之间用空行分隔：

```markdown
这是第一段。

这是第二段，和上面隔了一个空行。
```

### 文字样式

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***加粗斜体***
~~删除线~~
`行内代码`
```

### 列表

无序列表：

```markdown
- 苹果
- 香蕉
  - 次级项目（缩进两个空格）
- 橙子
```

有序列表：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

### 链接与图片

```markdown
[描述文字](https://example.com)

![替代文字](https://example.com/image.jpg)
```

### 引用

```markdown
> 这是一段引用
> 可以跨多行
>
> > 引用里可以嵌套引用
```

### 代码块

用三个反引号包围，可以指定语言实现语法高亮：

```python
def hello():
    print("Hello, World!")
```

### 分割线

三个或以上的 `-`、`*` 或 `_`：

```
---
```

## 进阶用法

### 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
```

### 表格

```markdown
| 名称 | 价格 | 库存 |
|------|------|------|
| 苹果 | 5 元 | 100 |
| 香蕉 | 3 元 | 200 |
```

### 脚注

```markdown
这里需要注释[^1]。

[^1]: 这是脚注内容。
```

## 在 Hugo 中写 Markdown

Hugo 使用 Goldmark 渲染 Markdown，支持标准语法和一些扩展功能：

- **Shortcodes**：可以嵌入视频、音乐等复杂元素
- **Front matter**：文章头部的元数据区
- **语法高亮**：支持几十种编程语言

文章头部示例：

```yaml
---
title: "文章标题"
date: 2026-06-13
tags: ["tag1", "tag2"]
description: "文章摘要"
---
```

## 结语

Markdown 的核心哲学是**「内容优先」**。花五分钟学会它，写文档能省下无数时间。写博客也好，记笔记也好，把格式交给工具，把创造力留给自己。
