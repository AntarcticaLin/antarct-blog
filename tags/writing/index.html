Markdown 写作指南

Markdown 是一种轻量级标记语言，让你用纯文本格式写文档，却能输出格式丰富的 HTML。写博客、记笔记、写 README，Markdown 都是不二之选。 为什么推荐 Markdown？ 专注内容：不用分心调整格式，写就是了 纯文本：任何编辑器都能打开，永远不会过时 生态丰富：GitHub、Notion、Obsidian 等都支持 学习成本低：五分钟就能上手 基本语法 标题 # 一级标题 ## 二级标题 ### 三级标题 #### 四级标题 一个文档建议只有一个一级标题，二号标题用于分隔章节。 段落与换行 段落之间用空行分隔： 这是第一段。 这是第二段，和上面隔了一个空行。 文字样式 *斜体* 或 _斜体_ **粗体** 或 __粗体__ ***加粗斜体*** ~~删除线~~ `行内代码` 渲染效果：斜体、粗体、加粗斜体、删除线、行内代码。 列表 无序列表： - 苹果 - 香蕉 - sub item 缩进两个空格 - 橙子 有序列表： 1. 第一步 2. 第二步 3. 第三步 链接 [描述文字](https://example.com) [带标题的链接](https://example.com "鼠标悬停时显示的文字") 图片 ![替代文字](https://example.com/image.jpg) 引用 > 这是一段引用 > 可以跨多行 > > > 引用里可以嵌套引用 代码块 用三个反引号包围，可以指定语言实现语法高亮： ...
